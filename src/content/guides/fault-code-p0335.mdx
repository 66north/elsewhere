P0335 means the ECU is not receiving a valid signal from the crankshaft position sensor (CKP). The CKP tells the ECU the exact rotational position and speed of the crankshaft, which is essential for calculating ignition timing and fuel injection. Without it, the engine will not start or will stall immediately. This is a critical code — the vehicle will typically not run.

## Symptoms

- Engine cranks but will not start (most common)
- Engine starts then immediately stalls
- Check Engine light illuminated
- No-start condition with no other obvious cause
- Intermittent stalling at speed (sensor intermittently losing signal)

## Tips

- P0335 almost always means a no-start. If the engine is running and this code appears, it is likely intermittent — diagnose the wiring first.
- The CKP sensor is a wear item on high-mileage vehicles. Replace it proactively if other sensors on the harness are showing age.
- Wiring harness damage near the crankshaft pulley is common — the lower engine gets heat cycling and vibration that degrades insulation.
- After replacement, clear the code and test-drive over 20 km including highway speeds before confirming the repair.
