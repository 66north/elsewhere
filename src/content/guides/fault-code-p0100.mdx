Code P0100 indicates the engine control unit (ECU) has detected a problem with the Mass Air Flow (MAF) sensor circuit. The MAF sensor measures incoming air volume and helps the ECU calculate the correct fuel mixture. A faulty MAF sensor causes poor fuel economy, hesitation, and rough idle. This code is specific to petrol engines (6G75); diesel engines (4M41) do not have MAF sensors.

## Symptoms

- Check Engine light illuminated
- Rough idle (engine shakes at stoplight)
- Hesitation when accelerating
- Reduced fuel economy (10-20% worse)
- Engine stumbles or stalls at low RPM
- Black smoke from exhaust (overfueling)
- Loss of power under load

## Tips

- Always replace the engine air filter when diagnosed with P0100. A clogged filter commonly triggers this code.
- Do NOT use carburator cleaner or compressed air on the MAF sensor — this damages the sensitive heating element. Use only MAF-specific cleaner.
- MAF sensor contamination is often caused by a failing or clogged air filter. After cleaning the MAF sensor, replace the air filter to prevent recurrence.
- If the code returns after cleaning, the sensor is likely failing and needs replacement.
- A vacuum leak (cracked hose or loose connector) can also trigger this code by allowing unmeasured air into the engine. Inspect all air intake hoses.
